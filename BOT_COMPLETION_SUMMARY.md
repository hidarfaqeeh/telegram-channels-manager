## 📋 كيفية البدء

### 1. الإعداد الأولي
```bash
# تثبيت المتطلبات
pip install -r requirements.txt

# إعداد البيئة
cp .env.example .env
# عدل ملف .env بقيمك الحقيقية

# إعداد البوت
python setup_bot.py
```

### 2. تشغيل البوت
```bash
# الطريقة المفضلة
python start_bot.py

# أو مباشرة
python main.py
```

### 3. استخدام بوت التحكم
- أرسل `/start` لبوت التحكم
- استخدم الأوامر لإدارة القنوات
- راقب الإحصائيات والنشاط

## ⚙️ التكوين المطلوب

في ملف `.env` عليك تعبئة:
```env
API_ID=             # من my.telegram.org
API_HASH=           # من my.telegram.org  
BOT_TOKEN=          # من @BotFather
CONTROL_BOT_TOKEN=  # من @BotFather (بوت منفصل)
PHONE_NUMBER=       # رقم هاتفك
ADMIN_IDS=          # معرف المشرفين
```
